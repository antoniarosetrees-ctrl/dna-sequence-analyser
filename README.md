# DNA Sequence Analyzer

This is a simple bioinformatics project built using Python.

## Features
- Counts nucleotides (A, T, G, C)
- Calculates GC content
- Handles invalid input

## How to run
1. Install Python
2. Run:
   python main.py

## Example
Input: ATGCGCTA  
Output: GC Content: XX%
